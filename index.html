OSINT X – Privacy Policy

Last updated: <2025-10-28>

OSINT X is a simple Android app that helps you view OSINT-related posts from X (Twitter) and apply basic heuristics (risk color and confidence score). We designed the app to be privacy-first.

Data We Collect

  • On-device only: Your topics, saved accounts, risk labels, flags, and optional notes are stored locally on your device.
  • API token: Your X API bearer token is stored locally using encrypted storage. It is not sent to any server controlled by us.
  • Network requests: The app calls the official X API directly from your device to fetch public content.

What We Don’t Collect

  • No account registration with us
  • No analytics SDKs or tracking
  • No data selling or sharing

Permissions

The app requires the Internet permission to call the X API. No other dangerous permissions are requested.

Data Control

  • You can edit or delete topics and accounts at any time.
  • You can clear the stored API token from the in-app Settings.
  • Uninstalling the app removes all locally stored data.

Security

We use Android’s encrypted storage for the API token. No app data is transmitted to our servers.

Third-Party Services

Requests to the X API are subject to X’s Terms and Privacy Policy. Your use of the X API is governed by X.
